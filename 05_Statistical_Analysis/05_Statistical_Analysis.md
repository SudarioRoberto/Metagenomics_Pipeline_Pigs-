# 05 - Statistical Analysis

This step performs statistical analysis on the taxonomic abundance data generated from Kraken2/Bracken.

## Overview

- Alpha and beta diversity analysis
- Differential abundance testing (ALDEx2, ANCOM-BC, MaAsLin2)
- Visualization of microbial communities

## Files

- `complex_diets.qmd` - Quarto source file
- `complex_diets.html` - Rendered HTML report
- `meta.csv` - Sample metadata
- `merged_*_abundance.txt` - Taxonomic abundance tables
